# Public Assets for Expo Web

This folder contains static assets that are served directly by Expo Web.

## Contents

- **TravalPass.mp4** - Background video for the landing page hero section (3.3MB)

## Usage

Files in this directory are accessible at the root path when running `expo start --web`.

For example:
- `/TravalPass.mp4` is accessible at `http://localhost:8081/TravalPass.mp4`

## Notes

- These assets are only used for web platform
- Keep file sizes reasonable to ensure fast page load times
- The video is sourced from the PWA project: `voyager-pwa/public/TravalPass.mp4`
